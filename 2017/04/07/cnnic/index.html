网事并不如烟：中文互联网流氓始祖CNNIC

发表于 2017-04-07 | 分类于 原理简介 | | 次阅读

1.先有鸿祎后有天，CNNIC还在前

  遥想新世纪初年，有个叫周鸿祎的人搞了一个叫“3721上网助手”的软件，为广大拼写字母有困难的网民带来了福音：“乡亲们，还在为记不住China mobile之类的鸟文烦恼么？快来装上3721吧，以后只要在地址栏输入‘中国移动’，就直达中国移动的网站啦！”

  感激涕零的乡亲们赶紧下载了安装包及其捆绑的N个小伙伴，从此过上了和流氓软件天天见的幸福生活。虽然也有不想领教的人，但 “上网助手”为乡亲们服务那是不管三七二十一的。每“服务”一台机器，便可从广告商处获得一份提成，也无怪乎周某人要不择手段地提高“市场占有率”了。后来眼见流氓同行越来越多，3721（已经换过“雅虎助手”、“奇虎360”、“360卫士”等许多马甲）便摇身一变，凭着丰富的流氓经验干起了收拾小流氓的营生，再次赢得了不少乡亲的赞许，走上了“杀人放火受招安”的青云大道。

  不过，本文的主角并不是这位招了安的“流氓软件之父”，而是正儿八经的衙门：中国互联网信息中心（China Internet Network Information Center，缩写为CNNIC）。强如“流氓软件之父”，在和这个衙门过招的时候也讨不了便宜。通过这篇介绍，乡亲们或许能够更好地了解，为什么360和百度这样的公司会出现在中国互联网这片神奇的土地上。

2.八字朝南开，财源滚滚来：

  CNNIC究竟是个啥呢，先引用一段世纪初的一篇报道《CNNIC的手》：

  “CNNIC 说：
中国互联网络信息中心(CNNIC)是成立于1997年6月3日的非盈利管理与服务机构，行使国家互联网络信息中心的职责。
  中央编委说：
不知道 CNNIC 是什么组织，但它肯定不是事业单位。
第一，它不符合《中华人民共和国国务院令1998年252号事业单位登记管理暂行条例》所规定的登记条件；
第二，它从来也没有向中央编委提出过登记申请。
  工商部门说：
不知道 CNNIC 是什么组织，但肯定不是企业。
它开不出发票，如果它开了发票，那肯定是在非法无照经营。
  民政部门说：
不知道 CNNIC 是什么组织，但肯定不是社团。

  CNNIC 在不是企业、不是事业、不是社团的情况下，已经经营了4年多。截至2001年7月，它已卖掉12万个cn域名。仅此一项，CNNIC 当年的毛收入就可以达到近4000万元，而且这个数字还在飞速增长之中。另外，CNNIC 还在做中文域名注册、流量认证、通用网址注册、认证培训等十分赚钱的业务。”

  可见，CNNIC从成立之始就既当裁判员，又当运动员（域名管理机构兼域名注册商）；既当婊子又立牌坊（打着“非营利机构”的名头闷声大发财——还不用上税）。说句题外话，具有同样特点的机构其实还很不少，其中最著名的大概是中国足协。

  很多年以前，CNNIC 就开始推行中文上网软件（“中文上网官方版软件”），风靡一时，以至于当时在 google里面输入“CNNIC”，搜索框的自动提示里，有一半是关于“卸载/专杀 CNNIC中文上网软件”的。该软件安装时悄无声息，卸载却需要十几个复杂的步骤，曾在2008年前后引起广泛的社会争议，一度被北京市网络行业协会列入10大流氓软件名单并勒令整改。当然，不难想到，CNNIC的应对措施是起诉把它列为流氓软件的杀毒软件制作者。从CNNIC的上级——工业和信息化部（简称“工信部”）强推“绿坝”监控软件惨遭失败，到CNNIC的下属3721被千夫所指，中国互联网史上的这一连串笑柄很可能不是巧合。

  那么在衙门的本职工作，也就是域名注册管理方面，CNNIC做得怎么样呢？一般网民大概都能感受到，09年以后，以.cn结尾的域名大幅度减少了。这不是错觉。07年前后的一段时间，大概是出于政绩的压力，CNNIC 以超低价格（“一元钱注册域名”等活动）显著提高了.cn 域名的注册量，一度使.cn 成为全球注册量最大的国别域名。但这种求量不求质的行为，必然导大量骗子网站、钓鱼网站、木马网站都用.cn 域名（因为可以降低诈骗成本中的注册费用），拖累了所有的.cn网站。2009年12月，CCTV点名批评 CNNIC 对域名的监管不力，导致大量黄色网站利用“cn 域名”逃避监管，吓得CNNIC赶紧发出通知，要求注册“cn 域名”必须提供“企业营业执照公章”。这等于变相阻止个人注册域名。那么占.cn域名总数量80%的个人注册域名怎么办？衙门回答“.cn 域名从未对个人开放注册过”。为了避免个人域名被强行收回，个人网站的站长们纷纷转投国外的“.com 域名”。在CNNIC发出通知后的短短一个月内，注销.cn域名的网站达到了百万数量级。此消彼长，.cn的网站注册数量很快跌出了全球三甲。

3.铁打的生意，流水的马甲——证书风波

  鉴于衙门高高在上的特殊地位，它的作恶能量远非360等流氓个体户可比。在介绍令CNNIC“声名大噪”的证书风波之前，有必要简单讲一讲数字证书的重要性。如今越来越多的网站地址开头不是“http://”而是“https://” 。这一“s”之差可不简单，因为http协议是明文传输的，你在网页上输入了什么密码、点击了什么按钮，在稍懂网络技术的人眼里都是透明的；而https协议则是加密的，可以保证信息传输的安全。

  那么，我们如何知道一个网站是真正加了密的而不是伪装加密来骗人的呢？这就需要权威机构认证并颁发 “数字证书”。这种权威机构叫“证书授权中心”（Certificate Authority，缩写为CA）。可见证书授权中心是https加密传输系统的核心，一旦授权中心出了问题，整个加密通讯都会跟着变得不安全。

  CNNIC明面上毕竟穿着“中国互联网络信息中心”这一身官衣，它制作的数字证书在2009年底相继得到了Mozilla（火狐浏览器的开发公司）和微软公司的承认。这一来，藏身于CNNIC卵翼之下的众多流氓厂商立刻获得了合法身份，不再会被浏览器警告安全风险了。在随后的几年中，CNNIC滥发证书、管理混乱的情形愈发严重，恶评如潮，终于被微软、Google、苹果在内的几大操作系统和浏览器厂商吊销了证书授权中心资格。2015年以后，连 CNNIC 自己的官网也不再使用自己的证书了。

  不过，向流氓厂商卖证书毕竟是门好生意。经过这次灰头土脸的失败，CNNIC不再赤膊上阵，而是在2016年让360的某个子公司出面，披上一身马甲，名为“沃通”数字证书公司。 在短短的一年内，“沃通”已经展示出了违规操作、伪造日期、一证多发等强大的“血统”证明，又暗中收购了“StartCom”等知名国外证书授权中心，其志不小，我们还是小心为妙。
