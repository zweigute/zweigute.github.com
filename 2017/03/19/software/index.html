怎样安全下载软件

发表于 2017-03-19 | 分类于 实用技巧 | | 次阅读
“只是按了‘一键安装’，怎么桌面上就多了那么多奇怪的图标？”
“明明点了下载，怎么出来的东西货不对样板？”
“XX助手说我下载的软件有风险，该怎么办？”

  太阳底下，并无新事。这些问题从来都不是什么新困境，只是把我们生活中常见的情景搬到虚拟世界中，披上了一层高科技的外衣罢了。相信见识过以下情形的人都知道是怎么回事：
  “只是采了几个白蘑菇炖个汤喝，怎么全家都中毒了？”
  “明明说好的参观景点，怎么导游尽往特产商场走？”
  “XX保险公司的推销员说我存钱的银行有倒闭风险，该怎么办？”

  凡此种种，即使到了2017年，“野生蘑菇煮汤，全家就医急救”之类的新闻也没有绝迹。不过这样的事终究只能成为新闻，可见绝大多数人不必非得精通植物学、毒理学和医学才能幸免。同理，保障个人计算机的系统安全和网络安全，并不一定要求使用者对硬件、软件有多么精深的了解，首要的是破除那种对电子产品莫名其妙的恐惧感、敬畏感或是无条件信任感——因为创造它们、使用它们的都是人类。

软件安全：

  随着计算机与网络的发展，它们在我们身边扮演的角色也日渐变得重要，从“玩具”、“工具”变成了“秘书”、“管家”。不，甚至不是“管家”而是“家”。一个保管着你的相册（数码照片）、档案（电子通讯录）、甚至存折（网上银行）的地方，称其为一栋电子楼房并不为过。如果说CPU、硬盘、显示器是这幢房子的承重框架，操作系统是它的水电管线，那么打字、上网、看电影用到的各种应用软件就是大大小小的家具了。

  这个比喻虽然粗糙，但我们确实应当用挑选家具的标准去决定在自己的电脑里装什么软件。家具越多，房子空间越小；软件越多，电脑运行越慢。就像我们不会在家里装大功率高压汞灯、不会购买牛皮癣广告上的产品、更不会接受来路不明的人做家政和保安一样，我们也不应该随心所欲地下载、试用自己不需要的软件，更不能因为某些软件自称“安全卫士”、“软件管家”就放手让它管理自己的电子财产。

  把软件从服务器下载到自己的电脑里，看似只需要一次简单的搜索和点击，但无孔不入的流氓厂商会在这一过程中想尽办法钻进你的电脑并且赖着不走。以著名的百度卫士系列流氓软件（俗称“百度全家桶”）为例，这里的许多案例提到了篡改网址、诱导广告、捆绑安装包、静默安装、防止卸载等种种手段，无所不用其极，令人大开眼界。流氓软件除了大量占用内存拖慢电脑速度，还有扫描用户文件、窃取用户隐私的斑斑劣迹。其他的“全家桶”厂商，如360、瑞星、金山等也大同小异，彼此之间还有激烈的竞争。一旦用户中了全家桶，想请走这几尊瘟神就只能重装系统了。

  了解了流氓软件的行径，就不难得出安全下载软件的两条原则：
  1. 从安全的来源下载。尽量使用正版，尽量从官网下载软件。
  2. 在安全的环境中安装。一粒老鼠屎能毁掉整锅粥，电脑上有流氓软件存在，其他软件就没有安全可言。

  实际操作中，第一条涉及怎样搜索找到真正的官网。从上面百度全家桶案例里也能看到，流氓搜索引擎从入口开始就布下了天罗地网，每一步都防不胜防。篡改网址、把“李鬼网站”标记为“李逵官网”之类的只能算是小儿科手段。一般用户既没有精力，也没有必要和流氓斗法，使用谷歌、必应等其他搜索引擎要好得多。

  理论上，盗版软件都有潜在的安全风险，因为谁也不知道破解正版软件的人有没有在里面加入私货。但是在全民盗版时代享受过免费windows的用户，很少有为正版软件付费的自觉。在各种“免费杀毒”流氓软件的风潮过后，“免费的就是最贵的”观念开始深入人心，或许有一天，我们能够重新在国内看到清洁的互联网环境。
