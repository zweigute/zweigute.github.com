互联网的目录：搜索引擎

发表于 2017-03-11 | 分类于 原理简介 | | 次阅读

1.网络搜索的特点

  学习使用“搜索引擎”是每个人在网络上必修的第一堂课。我们对“搜索”并不陌生，因为在字典里查个生字、在货架上寻件商品、在大楼里找个房间都是我们日常生活中极其常见的行为，即便是小学生也能很快学会。这给了我们一种错觉，彷佛在网络上搜索自己需要的信息也是一件少有技术含量的简单工作。

  事实上，日常生活中的搜索之所以简单，是由于它们普遍具有搜索对象少、分类排序良好的特点。《康熙字典》记载了大约4万7千个条目，美国国会图书馆藏书约3000万种。这些数目当然不算小，但远在谷歌（google）等著名搜索引擎问世之前，互联网上的网页总数就已超过了千万数量级。此外，互联网的搜索是以网页中的内容为对象，相当于书本中的每一个字都在目录中有它的索引。想象一下这样一份目录的厚度，就不难理解互联网的搜索结果数量有多么惊人。因此，对数以百万级的搜索结果加以排序，把用户最可能需要的网页排在最前面，是搜索引擎必须解决的首要问题。

2.脱颖而出的排序算法

  互联网发展的早期，搜索引擎的排序思路比较简单，以被搜索词语在网页中出现的次数多寡来排序。理论上，一个关键词在某个网页上出现得越多，越有可能意味着这个网页就是搜索者需要的结果。然而，这也同样意味着，广告网页和垃圾网页只要在空白处塞满各种常见关键词，就能获得很高的搜索排名，最终导致搜索结果的前几名乃至前几十名都被广告和恶意网页占领。

  正是在这种困境下，美国斯坦福大学的两位研究生佩奇 (Larry Page) 和布林 (Sergey Brin) 走出了一条新路。他们借鉴了学术界评判一篇论文的重要性时采用的标准，即“被引用次数”。简单来说，一篇论文被其他人引用，说明它受到了同行的认可和推荐；若是被业内泰斗引用，那么即使作者是个名不见经传的新人，也会受到重视。同样的，一个网页被其他网页链接，是一种远胜于“自荐”的“他荐”，若是被知名大网站链接，则排名还应当再高些。如此一来，要知道一个网页的排名，需要先知道链接到它的那些网页的排名，层层递推，最后搜索结果的排序问题就转化为对整个互联网链接结构的研究，成了一个不折不扣的抽象数学问题。

  解答这个数学问题需要对随机过程理论有一定了解，此处就不作展开了。从结论上讲，这个数学问题的答案提供了一种严谨合理的网页搜索结果排序方法，使得含有优质内容的网页能够确保排名靠前，而弄虚作假者必须费尽心机到各大论坛上发垃圾广告帖子，再链接到自己的恶意网页，才能在搜索结果中勉强往前挤一挤。此外，排序结果只与互联网链接结构有关，而与用户输入的具体搜索内容无关，这意味着排序可以在任何时间独立进行，而不必等到用户按下搜索键之后才开始，由此可以大大提高搜索速度和用户体验。佩奇将自己的这一研究成果双关地命名为“佩奇排序”（Page Rank），因为他的姓氏佩奇“Page”在英语中也有“网页”的含义。

3.结语

  佩奇和布林在1998年创立了谷歌公司，只用了六年就在搜索引擎界确立了无可动摇的地位。如今它的搜索范围已经不限于文字，还扩展到图片、音乐等其他媒体形式，深入到互联网生活的方方面面。它的发展壮大又一次印证了，在互联网上，知识能以最快的速度转化为力量。
